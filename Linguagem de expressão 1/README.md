# LE1 - Projeto Maven para NetBeans

Este projeto contém a implementação Java da Linguagem de Expressão 1 (LE1), organizada para abrir diretamente no NetBeans.

## Como abrir

1. Extraia o ZIP.
2. No NetBeans, vá em `File > Open Project`.
3. Selecione a pasta `LE1-NetBeans`, onde está o arquivo `pom.xml`.
4. Clique em `Open Project`.
5. Abra `src/main/java/le1/Main.java`.
6. Clique em `Run Project`.

## Observação

Os arquivos originais foram convertidos para UTF-8 para evitar erro de encoding no NetBeans/JDK moderno.
